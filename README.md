# Calculadora Web Seccionada

Aplicación web de calculadora básica desarrollada con HTML, CSS y JavaScript. Presenta una interfaz dividida en secciones que permite visualizar claramente cada parte de la operación: primer número, operador, segundo número y resultado.

---

## Características

* Operaciones básicas:

  * Suma
  * Resta
  * Multiplicación
  * División

* Interfaz estructurada por secciones

* Entrada de datos:

  * Mediante botones en pantalla
  * Mediante escritura directa en los campos

* Funciones adicionales:

  * Limpieza de datos
  * Eliminación del último carácter

---

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

---



## Uso

1. Clonar el repositorio:

```
git clone https://github.com/tu-usuario/tu-repo.git
```

2. Abrir el archivo `calculadora.html` en un navegador.

3. Utilizar la calculadora ingresando valores mediante los botones o el teclado.
